How to Run
  pip install -r requirements.txt
  streamlit run app.py

Project Overview
  This project focuses on analyzing global earthquake data to identify seismic patterns, risk zones, and trends. 
Using API-based retrieval from the USGS, 
the system processes and stores data in a MySQL database to perform advanced analytics and visualize insights through an interactive Streamlit dashboard

Skills & Technologies
  Language: Python (Pandas, Requests, Regex) 
  Database: MySQL & SQLAlchemy 
  Visualization: Plotly & Streamlit 
  Domain: Seismology & Disaster Management
